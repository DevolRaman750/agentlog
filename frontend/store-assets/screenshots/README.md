# App Store Screenshots Requirements

## iOS App Store Screenshots

### iPhone Screenshots:
- **iPhone 6.7"** (iPhone 15 Pro Max, 14 Pro Max, 13 Pro Max, 12 Pro Max): 1290x2796px
- **iPhone 6.1"** (iPhone 15, 14, 13, 12): 1179x2556px
- **iPhone 5.5"** (iPhone 8 Plus): 1242x2208px

### iPad Screenshots:
- **iPad Pro (6th gen) 12.9"**: 2048x2732px
- **iPad Pro (4th gen) 11"**: 1640x2360px

### Requirements:
- 3-10 screenshots per device type
- Show key features and user flows
- High quality, no pixelation
- Use actual device frames (optional but recommended)

## Android Play Store Screenshots

### Phone Screenshots:
- **Minimum**: 320px
- **Maximum**: 3840px
- **Recommended**: 1080x1920px (16:9 ratio)

### Tablet Screenshots:
- **7-inch**: 1024x600px or 1024x768px
- **10-inch**: 1280x800px or 1280x1024px

### Requirements:
- 2-8 screenshots
- JPG or 24-bit PNG (no alpha)
- Landscape and portrait orientations

## Screenshot Strategy for AgentLog:

1. **Onboarding/Welcome Screen**
   - Show clean, professional interface
   - Highlight AI agent management

2. **Agent Dashboard**
   - Display multiple agents
   - Show status indicators and metrics

3. **Agent Configuration**
   - Demonstrate easy setup process
   - Show customization options

4. **Execution Results**
   - Display successful agent runs
   - Show clear output and logs

5. **Settings/API Configuration**
   - Show professional configuration options
   - Highlight security features
